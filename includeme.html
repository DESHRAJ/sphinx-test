JOBS¶

Creating Model¶

For creating a new Model, a Dataset is required. On the right side of home page, under New Models, there is a button Images. Choose Classification from the dropdown list that is shown by clicking on Images Button. After this, you will be redirected to the url /models/images/classification/new. Now, select one of the Dataset, choose one of the Networks and you can change the parameters in the Solver Options. Also, you can create a custom network yourself using a .prototxt file.

You can use the image_classification_model_create() method from the file digits/model/images/classification/views.py@49 to achieve this task.

Delete Job¶

A previously created job (dataset/model) can deleted (or even aborted in the case of a model) using DIGITS. Simply click on the delete button associated with the DatasetJob or ModelJob.

Classify a Model¶

To classify a Model, goto the homepage of DIGITS and click on one of the Model under the completed category. It will take you to the link /models/<job-id> (digits/model/views.py@27) where at the bottom of the HTML page, there are lot of options for classifying.

For classifying a Model, DIGITS provides the functionality to classify using Single Image, multiple images, using a .txt filedigits/templates/models/images/classification/show.html@160) or through a URL of an image or set of Images. Select the way in which you want to classify and click the classify button and here you go. You will see a list of classification results according to the images supplied and their percentage predictions. The respective methods for these options of classification can be found as follows :

image_classification_model_classify_many() (digits/model/images/classification/views.py@231) image_classification_model_classify_one() (digits/model/images/classification/views.py@231)

You can also view the outputs (ROC curves) of classification by calling the image_classification_model_large_graph() method from the file (digits/model/images/classification/views.py@218). The final outputs would look something like the images below.

The TOP-N classification of images is also supported by DIGITS and can be accessed by calling the image_classification_model_top_n() method from (digits/model/images/classification/views.py@364)

The visualization of the learning rate graph can be acquired by calling the models_visualize_lr() method from (digits/model/views.py@99)

You can obtain a visualization of the neural network that you use for creating your model by using the models_visualize_network() method from (digits/model/views.py@86)
